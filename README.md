## Getting Started

Clone the GIT repository

```
git clone https://github.com/lncm/satoshi-pi-client
```

Install dependencies

```
yarn --ignore-engines
```

Run the app

```
yarn start
yarn start:native
```

How to lint code?

```
yarn lint
```


How to test code?

```
yarn test
```

How to release a new v{mayor}.{minor}.{patch}?

```
yarn release
```
